Introducción

Aprenderemos sobre los conceptos básicos de HTML y CSS, los dos componentes básicos de casi todo en la web.



Qué es HTML?

En ingles HTML es Hyper Text Markup Language y en español significa lenguaje de marcado de hipertexto

Dictionary.com define marcado como


un conjunto de instrucciones detalladas, generalmente escritas en un manuscrito que se va a componer, relacionadas con el estilo de la tipografía, la composición de las páginas y cosas por el estilo. Entonces, puede pensar en HTML como el lenguaje utilizado para crear instrucciones detalladas sobre el estilo, el tipo, el formato, la estructura y la composición de una página web antes de que se imprima (que se le muestra).

Pero en el contexto del desarrollo web, podemos reemplazar el término "impreso" por "renderizado" como un término más preciso. HTML lo ayuda a estructurar su página en elementos como párrafos, secciones, encabezados, barras de navegación, etc.

Así es como puede formatear y estructurar un documento con solo HTML. Como puede ver, este marcado contiene algunos elementos web como:
  • Encabezado de nivel 1 h1
  • Encabezado de nivel 2 h2
  • Encabezado de nivel 3 h3
  • Párrafo p
  • Una lista no ordenada ul li
  • El cuerpo de la página body
