Fundamentos de HTML

HTML y CSS son dos lenguajes que trabajan juntos para crear todo lo que ves cuando miras algo en Internet. HTML son los datos sin procesar a partir de los cuales se construye una página web. Todo el texto, enlaces, tarjetas, listas y botones se crean en HTML. CSS es lo que agrega estilo a esos elementos simples. HTML pone información en una página web y CSS le coloca estilo.
